# Korean Tutor Online 🇰🇷

Онлайн-версия системы учёта оплат на GitHub Pages + Supabase.

## Как запустить

### 1. Загрузи на GitHub
Создай репозиторий (например `korean-tutor`) и загрузи `index.html`.

### 2. Включи GitHub Pages
Репозиторий → Settings → Pages → Source: `main` branch → `/root` → Save

Через 1-2 минуты система будет доступна по адресу:
```
https://ТВО_ЛОГИН.github.io/korean-tutor
```

### 3. При первом открытии
Система попросит ввести данные Supabase (URL + anon key).
Найти их: supabase.com → твой проект → Settings → API

### 4. Перенести данные из старой версии
Поскольку раньше данные хранились в браузере (localStorage),
их нужно внести заново через интерфейс новой системы.

## Связь с ботом
Бот (korean-tutor-bot) и эта система используют одну базу Supabase —
данные синхронизированы автоматически в реальном времени.
